# 📋 Settings Manager

A simple Python project to manage application settings using dictionary operations. Supports adding, updating, deleting, and viewing settings.

## 🌟 Features

- ✅ Add new settings
- ✅ Update existing settings
- ✅ Delete settings
- ✅ View all settings
- ✅ Case-insensitive keys and values
- ✅ Input validation
- ✅ User-friendly messages
